Step 1

Choose What To Include

The main library is included by default. Add learning tools if you want extra offline education content on this device.

The main library stays selected because it is the base of SOPR. Learning tools can be added now or later.

Step 2

Choose Your Wikipedia Size

Choose the Wikipedia option that best fits your storage space and how much information you want available offline.

Wikipedia choices will appear here once setup information loads.

Step 3

Choose Your Map Regions

Select one or more offline map regions to install now. You can choose all regions, or just the areas you want on this device.

Map choices will appear here once setup information loads.

Step 4

Choose How People Connect

Use an existing network for the simplest setup, or let this device create its own Wi-Fi so phones and laptops can connect directly.

Step 5

Review Your Choices

Complete setup to download content, turn on services, and finish preparing this device.
