Initial Setup

Easy Setup Wizard

Move step by step, choose the modules you want, pick a Wikipedia package, and review everything before downloads and services begin.

Step 1

Select What To Install

Knowledge Library is included by default. Add the Education Module if you want Kolibri and structured offline learning tools on this device.

Knowledge Library stays selected because it is the base of the SOPR appliance. The Education Module can be added now or later.

Step 2

Choose Your Wikipedia Level

Select the package that fits your storage budget and expected use. These options are loaded directly from the project JSON so they can be updated without editing the page.

Wikipedia package details will appear here once the setup data loads.

Step 3

Choose How People Connect

Use an existing network for simpler installs, or enable the onboard access point so phones and laptops can join the device directly.

Step 4

Review Your Setup

Save your preferences if you want to come back later, or apply the configuration now to run downloads, enable services, and finish setup.
