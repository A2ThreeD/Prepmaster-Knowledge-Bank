Prepmaster Admin

This is the custom admin placeholder for system controls, content updates, and maintenance actions.

Settings

The settings area should manage device-level options such as hostname, content paths, and service preferences.

Reopen the setup workflow at any time to change install options, Wikipedia package selection, or other first-start preferences.

Wireless Access Point

Configure and enable direct Wi-Fi access so phones, tablets, and laptops can join the Pi and reach local content without another router.

Use PREPMASTER_AP_ENABLED=1 in config/prepmaster.env, then run sudo ./scripts/configure_access_point.sh.
